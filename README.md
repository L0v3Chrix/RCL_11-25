# Recovery Centered Living - Website Redesign

**Project Start Date:** October 21, 2025
**Client:** Recovery Centered Living (RCL)
**Founders:** Slade & Chloe Skaggs
**Location:** Austin, Texas

---

## Project Overview

Complete website redesign and platform development for Recovery Centered Living, a network of six sober living homes in Austin, Texas. This project aims to create a flagship site that can be white-labeled and resold to other sober living providers nationwide.
